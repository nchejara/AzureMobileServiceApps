Application Usage ...
   1. Create an appication on windows Azure mobile service and add three tables
      [ Customers, Friends and Checkins ].
   2. Add Application URL and Application Key in given text box.
   3. Click on Insert Button.
   4. Make sure all the data are inserted.
   5. After insert all data successfully, Go to windows Azure and change all tables
       permission as Autintication User.
   6. Run WhereTheFriend application and enjoy the UI.
